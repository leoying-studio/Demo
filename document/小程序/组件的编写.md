# 小程序组件的编写方式
> 使用template的方式
 - 组件的定义
	```
		// article组件
		<template name="article">
			<view>{{title}}</view>
			<view>{{body}}</view>
			<view>{{footer}}</view>
		<template>
	```
 - 组件的使用
   ```
      // 在页面中去引用, 先去找到article组件
		<import src="./../component/article.wxml"/>
		// 开始使用, is属性可以指定article地址
		<template is="article" data={{{title:'张三', body: "你好", footer: "footer"}}}>
	  
   ```
